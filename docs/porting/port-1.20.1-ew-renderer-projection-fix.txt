## Renderer retarget

`EwSandstormOverlayRenderer` no longer draws a fixed `[-1, 1]` quad through the world render matrix and `ctx.consumers()`.

It now draws a framebuffer-sized translucent veil with an owned orthographic projection, owned model-view reset, depth writes disabled during the veil, and blend enabled with the default blend function.

The EW distance/math/color/alpha were preserved.

## Hotbar state correction

After manual proof showed the bottom HUD/hotbar area being obscured, the renderer cleanup now leaves blend enabled/defaulted for subsequent hand/HUD rendering instead of exiting with blend disabled.

The experimental HUD bottom-fill profile was rolled back to the prior bottom-safe fade so the HUD overlay does not cover the hotbar strip.
